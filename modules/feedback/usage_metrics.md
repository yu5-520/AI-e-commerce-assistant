# Usage Metrics

These metrics measure whether users feel the output is useful.

## Lightweight Metrics
- copied_count
- selected_count
- like_count
- rewrite_request_count
- follow_up_request_count
- rejection_count

## Section Metrics
Track usage by section:
- title_package
- image_package
- sku_suggestion
- price_plan
- operation_action

## Rule
Usage metrics show user experience, not real operation effect. Do not treat high usage as proven business success.
