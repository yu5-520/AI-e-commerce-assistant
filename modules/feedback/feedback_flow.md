# Feedback Flow

Goal: turn useful AI outputs into reusable knowledge.

## Flow
1. AI generates result card.
2. User selects, copies, likes, or rejects output sections.
3. User optionally marks whether the suggestion was executed.
4. User optionally submits operation effect metrics.
5. System classifies feedback level: L1/L2/L3/L4.
6. Qualified feedback enters candidate knowledge base.
7. Effective patterns become reusable knowledge for future retrieval.

## Principle
Do not force long feedback forms. Prefer quick choices first, detailed operation metrics second.
