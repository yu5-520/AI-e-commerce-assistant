# Feedback Schema

Feedback is collected after AI output and user action.

## Basic Fields
- platform
- mode
- category
- product
- result_section
- output_item
- feedback_level
- user_experience_score
- operation_effect_score
- user_note

## Usage Feedback
- copied
- liked
- selected_for_use
- asked_for_more
- rejected
- needs_rewrite

## Execution Feedback
- used_title
- used_image_structure
- changed_sku
- changed_price
- joined_activity
- used_ad_action

## Effect Feedback
- exposure_change
- click_rate_change
- conversion_change
- order_change
- profit_change
- refund_change
- inventory_clearance_change
