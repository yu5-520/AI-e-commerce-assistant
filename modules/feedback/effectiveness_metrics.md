# Effectiveness Metrics

These metrics measure whether the output helped real store operation.

## Core Metrics
- exposure_change
- click_rate_change
- conversion_rate_change
- order_count_change
- profit_change
- refund_rate_change
- inventory_clearance_speed
- activity_approval_result
- ad_roi_change

## Before / After
Each effective case should keep before and after values when possible.

## Rule
Operation effect is stronger than user preference. A suggestion can look good but still fail in store operation.
