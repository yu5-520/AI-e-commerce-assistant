# Embedding Schema

Only feedback-filtered knowledge should be embedded.

## Embeddable Knowledge Types
- title_pattern
- image_pattern
- sku_pattern
- price_pattern
- operation_case
- feedback_summary

## Embedding Text Should Include
- pattern type
- platform
- mode
- category
- product
- user goal
- reusable structure
- sample output if useful
- feedback level
- operation effect summary

## Do Not Embed By Default
- raw AI output with no feedback
- private user data
- full chat logs
- duplicate low-quality suggestions
- unverified claims

## Principle
Embedding text should describe why the pattern is reusable, not only what the generated text was.
