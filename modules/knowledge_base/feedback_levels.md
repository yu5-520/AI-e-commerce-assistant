# Feedback Levels

Do not treat every AI output as knowledge.

## L1 Generated
AI generated the content, but user has not reacted.
Storage: history only.

## L2 Liked
User marked it useful, copied it, or asked for more in the same direction.
Storage: candidate knowledge.

## L3 Executed
User actually used the title, image structure, SKU suggestion, price change, or operation action.
Storage: execution knowledge.

## L4 Effective
User reported better exposure, clicks, conversion, profit, activity result, or inventory clearance.
Storage: high-value reusable knowledge.

Key principle: user experience score and operation effect score must be tracked separately.
