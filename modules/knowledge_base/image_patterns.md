# Image Pattern Knowledge

Store reusable main-image structures that users like or that improve click rate.

Suggested fields:
- platform
- mode
- category
- image_type
- main_text_structure
- sub_text_structure
- visual_layout
- user_experience_score
- click_effect_score
- applicable_boundary

Common Pinduoduo image types:
- price benefit image
- function selling point image
- usage scenario image
- comparison or pain-point image
