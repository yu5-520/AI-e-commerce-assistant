# Operation Case Knowledge

Store validated operation cases, not every AI output.

Suggested fields:
- platform
- mode
- category
- product
- user_goal
- action_taken
- title_used
- image_structure_used
- sku_change
- price_change
- activity_or_ad_action
- before_metrics
- after_metrics
- user_experience_score
- operation_effect_score
- reusable_pattern
- applicable_boundary

Case levels:
- candidate: user liked the output
- executed: user used it in store operation
- effective: metrics improved after use
- reusable: pattern can be reused across similar products
