# 素材观察 Agent

## 定位

素材观察 Agent 是生成方案前的隐藏产品工程模块。它不直接面向普通用户展示，也不要求用户确认素材包。

用户侧只看到：

```text
正在整理商品素材 → 正在生成可测试方案 → 可复制结果
```

工程侧会在结果记录中看到素材观察输出，用于检查标题词感、时效和素材来源边界。

## 输入

- 商品词
- 运营模式
- 当前时间/季节
- 用户补充的竞品标题、主图卖点或平台热词
- 后续可接入商家自有数据、上传截图 OCR、合法搜索 API

## 输出

- 可用词感
- 标题结构
- 建议搜索任务
- 禁用词
- 风险标记
- 置信度
- 下一步采样建议
- Agent 运行轨迹

## 边界

本 Agent 不做未经授权的平台爬虫，不复制竞品标题原文，只提取词感、结构和风险。
