# 最终 Demo 叙事线

## 1. 不是 AI 文案工具

项目最初可以从商品标题、主图、SKU 入手，但最终升级方向不是“更会写文案”，而是“更懂业务流程”。

## 2. 不是无边界 Agent

电商 ERP / CRM / RPA 场景涉及真实资金、商品、客户和售后风险，因此不适合一开始就做无边界 Agent。

## 3. 是 Workflow-first AI 工作流

项目采用：

```text
ERP / CRM 数据
↓
RAG 知识依据
↓
AI 诊断
↓
RPA 任务草案
↓
人工确认
↓
日志回写
```

## 4. 已经有可运行骨架

当前可以运行：

```bash
python -m src.run_demo
```

也可以打开：

```text
web_demo/index.html
```

## 5. 价值总结

这个项目展示的是：

> 如何把 AI 安全地接入真实电商经营流程，而不是让 AI 无边界替人操作店铺。
