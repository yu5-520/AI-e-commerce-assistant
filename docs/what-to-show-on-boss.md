# BOSS 附件 / 作品展示建议

## 1. 展示顺序

建议展示 5 个点：

1. 项目首页定位：AI + RPA + ERP + CRM 电商经营自动化工作台。
2. 前端 Demo：导入 Mock 数据、AI 诊断、RPA 任务草案。
3. Python Workflow：能运行 `python -m src.run_demo`。
4. Evals：能运行 `python evals/run_evals.py`。
5. 风控边界：高风险动作必须人工确认。

## 2. 附件简历里写什么

重点写：

- Workflow-first
- ERP / CRM 数据建模
- RAG 知识增强
- RPA 任务草案
- Human-in-the-loop
- Evals
- 前端 Demo

## 3. 不要写什么

不要写：

- 已经接入真实 ERP
- 已经接入真实 CRM
- 已经自动运营店铺
- 已经自动投放广告
- 已经自动触达客户
- 已经是生产级系统

## 4. 推荐一句话

> 独立设计并搭建 Workflow-first 的电商 AI 自动化工作台 MVP，使用 Mock ERP / CRM 数据跑通商品诊断、客户分层、RAG 召回、RPA 任务草案、人工确认和报告输出闭环，并通过前端 Demo 与 Evals 展示业务流程和安全边界。
