# V6 面试安全表述

## 推荐说法

当前项目是一个可运行的 MVP 骨架，而不是生产级系统。

我先用 Mock ERP / CRM 数据和规则引擎跑通业务闭环，包括商品诊断、客户分层、RAG 召回、RPA 任务草案、人工确认和报告输出。

这样做是为了先验证业务流程、输入输出结构和安全边界。

后续可以逐步替换为真实 LLM、Embedding 向量库、ERP / CRM API 和 RPA 执行器。

## 不推荐说法

不要说：

- 这是完整企业系统
- 已经真实接入平台后台
- 已经能自动运营店铺
- 已经能自动触达客户
- 已经实现 Agent 自主执行

## 一句话

> 当前重点是验证 AI 工作流如何安全接入电商经营流程，而不是追求无边界自动化。
