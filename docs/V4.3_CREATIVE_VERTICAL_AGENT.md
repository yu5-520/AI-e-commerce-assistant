# V4.3 Creative Vertical Agent

V4.3 adds the vertical category creative Agent.

The product rule is:

```text
标题主图 Agent 不是素材生成器，而是类目表达策略生成器。
```

It generates title directions, main-image directions, selling-point order, and A/B test plans from product facts, category profile, platform rules, competitor signals, and RAG memory.

## 1. Flow

```text
product facts
↓
category profile
↓
platform expression rules
↓
competitor signals
↓
RAG creative / playbook memory
↓
title variants
↓
main-image directions
↓
selling-point order
↓
A/B test plan
↓
human confirmation
↓
optional creative test task
```

## 4. Platform rules

V4.3 starts with four expression-rule groups:

```text
淘宝：搜索关键词覆盖、材质、场景、规格可信度。
拼多多：到手价值、套装数量、价格带、耐用。
抖音小店：场景痛点、短句卖点、人群需求、即时改善。
通用：商品事实、核心卖点、使用场景、风险边界。
```

## 5. Safety boundary

The creative Agent can:

```text
generate title variants
generate main-image directions
rank selling points
generate A/B test plans
cite RAG references
create a creative test task draft
```

It cannot:

```text
directly publish products
directly change price
directly change ad spend
directly write to ERP / CRM / marketplace data
generate exaggerated claims
skip human confirmation
```
