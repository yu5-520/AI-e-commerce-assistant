# Demo 风险声明

## 1. Demo 性质

本项目当前为 MVP 演示项目，不是生产级电商系统。

当前只使用 Mock 数据，不连接真实商家后台，不处理真实客户数据。

## 2. 不提供能力

本项目不提供：

- 自动改价
- 自动上架
- 自动下架
- 自动投放
- 自动报名活动
- 自动群发客户
- 自动处理退款
- 绕过验证码
- 绕过平台风控
- 生成违规营销话术
- 保存真实客户隐私

## 3. 风险控制

所有中高风险任务默认：

```text
requires_approval = true
auto_execution_allowed = false
```

## 4. 设计目的

本项目用于展示 AI 工作流设计、ERP / CRM 数据建模、RAG 知识增强、RPA 任务草案和 Human-in-the-loop 风控机制。

## 5. 结论

该 Demo 的核心是展示可控自动化，而不是无约束自动化。
