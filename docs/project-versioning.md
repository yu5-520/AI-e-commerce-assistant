# 项目版本说明

## V1：AI 商品增长 Demo

核心能力：

- 商品输入
- 标题 / 主图 / SKU 建议
- 测试数据回填
- AI 复盘

## V2：AI + RAG 商品运营工作台

核心能力：

- 商品档案库
- 平台规则库
- 合规风控库
- 运营方法库
- RAG 召回设计

## V3：AI + RPA + ERP 电商运营自动化

核心能力：

- ERP 商品 / 订单 / 库存 / 退款数据
- 商品经营诊断
- RPA 任务草案
- 人工确认机制

## V4：AI + RPA + ERP + CRM 电商经营自动化

核心能力：

- CRM 客户 / 标签 / 互动数据
- 客户分层
- 售后归因
- 复购任务草案

## V5：可运行 Mock Workflow

核心能力：

- Python 运行主流程
- JSON / Markdown 输出
- 最小 Evals
- 简单 RAG 检索

## V6：前端三段式 Demo

核心能力：

- 静态前端页面
- 数据导入展示
- AI 诊断展示
- RPA 任务草案展示
- 人工确认项展示

## 下一版本：V7 API 可交互 Demo

计划能力：

- FastAPI 后端
- 前端 fetch 调用
- 任务确认 / 拒绝状态
