# 自然流执行包

## 结论
基于现有信息，直接生成第一轮自然流测试方案；不要只说“下一步”。

## 1. 拼多多标题测试包
必须输出10个拼多多风格标题，分成以下类型：
- 搜索词覆盖型
- 低价/价格感型
- 功能卖点型
- 场景人群型
- 长尾关键词型

每个标题后面用一句话说明适合测试什么。

## 2. 主图文案方向
必须输出3套主图方案：
- A版：价格/利益点
- B版：功能卖点
- C版：使用场景

每套包含主图大字、副文案、画面建议。

## 3. 价格测试建议
结合成本、售价、运费、库存，给出保守测试价、活动/券后测试价、止损提醒。

## 4. 观察指标
列出本轮需要记录的指标：曝光、点击、点击率、成交、转化、退款、竞品价格。

## 5. 还缺什么
用普通商家能看懂的话说明最多3个缺失信息；如果缺失信息不影响第一轮标题/主图测试，就先给方案，不要卡住输出。
