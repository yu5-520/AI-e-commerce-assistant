# 求职关键词映射

## 1. 项目关键词

- AI 工作流
- Workflow-first
- RPA 自动化
- ERP 数据分析
- CRM 客户运营
- RAG 知识增强
- Human-in-the-loop
- Evals 评测
- 数据建模
- 业务流程自动化
- 风险控制
- 日志回写
- Mock Workflow

## 2. 岗位关键词匹配

### AI 产品助理 / AI 产品经理助理

可匹配内容：

- AI 应用产品设计
- 业务流程拆解
- Prompt 模板设计
- RAG 架构设计
- 产品原型 Demo
- 数据闭环和反馈迭代

### RPA 产品 / 自动化流程助理

可匹配内容：

- RPA 任务草案
- 人工确认机制
- 自动化流程设计
- 高风险动作拦截
- 日志回写
- 流程可追溯

### 电商产品 / 电商运营数字化

可匹配内容：

- ERP 商品 / 订单 / 库存 / 退款数据
- SKU 利润测算
- 库存预警
- 活动准备
- 商品复盘

### CRM / 用户运营产品

可匹配内容：

- 客户分层
- RFM 分析
- 复购判断
- 流失预警
- 售后归因
- 客户触达边界

## 3. 简历关键词组合

推荐写：

```text
AI 工作流 / RAG / RPA / ERP / CRM / Human-in-the-loop / Evals / 数据建模 / 业务流程自动化
```

不要写过度：

```text
生产级 Agent 系统 / 完整企业级 ERP 系统 / 已接入真实客户数据 / 已自动运营店铺
```

## 4. 一句话项目定位

> 设计并搭建 Workflow-first 的电商 AI 自动化工作台 MVP，围绕 ERP / CRM 数据接入、RAG 知识增强、AI 诊断、RPA 任务草案、人工确认和 Evals 评测，跑通从 Mock 数据到复盘报告输出的业务闭环。
