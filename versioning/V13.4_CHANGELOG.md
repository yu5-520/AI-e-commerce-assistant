# V13.4 Task Pool Station

## 目标

新增任务入池站，把 V13.3 的任务快照转成可见任务池任务。

## 核心原则

任务池站只负责入池，不负责接收、派发、提交、复核、复盘或 RAG 回流。

## 可入池的任务快照

```text
create_task_snapshot
manager_review_required
```

## 不入池的判断记录

```text
observe_only
ignore_noise
```

它们保留为 Agent 判断记录，不制造任务噪音。

## 当前边界

V13.4 创建的是可见任务池任务，但任务仍处于：

```text
待接收
待拆分
```

后续接收、派发、提交、复核仍由后续生命周期站点处理：

```text
V13.5：Task Acceptance / Assignment Station
V13.6：Submission + Review Station
V13.7：Recap + RAG Feedback Station
```
